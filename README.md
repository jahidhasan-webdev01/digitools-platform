# 🚀 DigiTools Platform

## 📌 Project Overview

DigiTools Platform is a modern digital tools marketplace built with **React.js** where users can explore and purchase premium digital products such as design resources, templates, and productivity tools.

The platform provides a clean user interface, interactive product cards, and a fully functional cart system. Users can add products to their cart, remove them anytime, and complete checkout smoothly with real-time notifications.

The project focuses on **component-based architecture, responsive design, and interactive UI experience** using modern frontend technologies.

---

## 🛠️ Technologies Used

* **React.js** – For building the interactive user interface
* **Tailwind CSS** – For modern and responsive styling
* **DaisyUI** – Pre-built UI components
* **JavaScript (ES6+)** – Application logic and state management
* **React Toastify** – For displaying alert notifications
* **JSON** – For storing product data

---

## ✨ Key Features

### 🛒 Interactive Cart System

Users can easily add products to the cart, remove selected items, and proceed to checkout. The cart count updates instantly in the navbar.

### 🔔 Real-time Notifications

All cart actions such as **adding products, removing items, and checkout completion** trigger beautiful toast notifications using React-Toastify.

### 📱 Fully Responsive Design

The entire website is optimized for **mobile, tablet, and desktop devices**, ensuring a smooth user experience across all screen sizes.
